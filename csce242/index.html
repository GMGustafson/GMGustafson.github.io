Assignments

Assignment 1 - Basic HTML

This assignment was to get us use to using VScode and the layout of a HTML. Our assignment was to build a page of anything we wanted.

Assignment 2 - Basic CSS

This assignment was to get us comfortable with using CSS. Similiar to assignment 1, we built a page on anything we wanted but added CSS to it.

Assignment 3 - Page Layout

This assignment was to get us comfortable with flex and using USS. We were given a photo and had to try and recreate it.

Assignment 4 - Recreate CSS

This assignment was to get us to be able to recreate any page. Formatting the HTML and CSS on our own. We had to try and recreate the UofSC Graduate School Admissions Page.

Assignment 5 - Java Script

This assignment was to get us to be able to recreate a page given to us, working on how to implement a counter, a random image, and a slider move.

Assignment 6 - JS Conditionals

This assignment was to get us to be able to recreate a page given to us working on how to implement, showing on one exercise at a time. Those exercises include a picture chooser and a color slider.

Assignment 7 - Loops

This assignment was to get us to be able to recreate a page given to us working on how to implement showing stars inside a box and when you click on one it tells you what star it is.

Assignment 8 - Arrays

This assignment was to get us to be able to recreate a page given to us working on how to implement, when you click on a photo, a message populates.

Assignment 9 - Classes

This assignment was to get us to be able to recreate a page given to us. We had to be able to create it when you click on a dog a expanded view popped up with information.

Assignment 10 - Json Parsing

This assignment was to get us to be able to recreate a page given to us. We had to be able to create it when you hover over the icecream, the name pops up.

Part 2 of the Project was to make a Wireframe on Mockitt to come up with how we wanted the website to look.

Project 3 - HTML & CSS

Part 3 of the Project was to start building the HTML and CSS for our website. We had to replicate our Mockitt design.

Project 4 - Colors, Pictures, and Text

Part 4 of the Project was to continue working on building our website, adding more CSS, pictures and text.

Project 5

Part 5 of the project was to make changes to Part 4 based of the feedback we received.

Project 6

Part 6 of the project was to make more changes and to add a page using JSON files and style that page.
